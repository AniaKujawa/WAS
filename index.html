Total element width = width + left padding + right padding + left border + right border + left margin + right margin

Total element height = height + top padding + bottom padding + top border + bottom border + top margin + bottom margin

Można inaczej

* { box-sizing: border-box; }

Total element width = width + left margin + right margin

Total element height = height + top margin + bottom margin

Układ elementów

  • { display: inline } - nie zaczyna nowej linii, nie ustala się wysokości i szerokości
  • { display: block }- rozpoczyna nową linię, można ustalać wielkość (domyślnie przyjmuje całą szerokość)
  • { display: inline-block }- jak inline, ale można ustalać wielkość
  • { display: none } - usuwa element

Układ elementów 2

  • { position: absolute } - relatywnie do niestatycznego przodka
  • { position: relative } - relatywnie do swojej normalnej pozycji
  • { position: fixed } - relatywnie do okna przeglądarki
  • { position: sticky} - relatywnie do scroll
  • { position: static} - domyślnie, elementy po kolei
  • {float: right || left }

  • { align-items: flex-start || flex-end || center || baseline || stretch }
  • { flex-direction: row || column || row-reverse || column-reverse}
  • { align-self: flex-start || flex-end || center || baseline || stretch}
  • { flex-wrap: nowrap || wrap || wrap-reverse }
